# Stomatal Conductance Dynamic Model

This repository contains the computational implementation of a dynamic model for estimating stomatal conductance from operationally accessible environmental variables, including radiation, leaf temperature, and leaf-to-air vapor pressure deficit.

## Purpose

The code reproduces the model calibration, simulation, validation, and graphical outputs associated with the manuscript submitted to *Environmental Modelling & Software*.

## Repository structure

- `stomatal_conductance_modelFR.ipynb`: Main Jupyter Notebook used for model implementation, calibration, simulation, validation, and figure generation.
- `requirements.txt`: Python package dependencies required to run the notebook.
- `LICENSE`: Software license.
- `CITATION.cff`: Citation metadata for the software.
- `data/`: Input data or example data required to reproduce the workflow.
- `results/`: Output files generated by the model.
